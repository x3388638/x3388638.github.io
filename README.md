# 2yc.tw

## Dev

### Setup

```
npm ci
```

Create github access token for reading pubilc repo and create file `.github_access_token` in app folder

```
module.exports = 'ACCESS_TOKEN'
```

**Run dev server**

```
npm run dev
```

**Build**

```
npm run build
```

**Deploy to gh-pages (master branch)**

```
npm run deploy
```
